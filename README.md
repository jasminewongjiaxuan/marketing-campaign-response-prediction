# Marketing Campaign Response Prediction

This project applies supervised machine learning models using Python to predict customer response to marketing campaigns.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Project Objectives
- Predict customer response to marketing campaigns
- Compare machine learning model performance
- Improve targeting accuracy for marketing strategies

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- Naive Bayes
- XGBoost

## Evaluation Metrics
- ROC-AUC
- Accuracy
- Precision
- Recall
